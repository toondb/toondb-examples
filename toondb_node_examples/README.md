# ToonDB Node.js SDK Examples

This directory contains examples for using the ToonDB Node.js SDK.

## Prerequisites

- [Node.js](https://nodejs.org/) 18+ installed.
- ToonDB Node SDK: `@sushanth/toondb`

## Setup

Install dependencies:

```bash
npm install
```

## Examples

### 1. Basic Key-Value Operations
Demonstrates basic PUT, GET, and SCAN operations.

```bash
node basic_kv.js
```

### 2. SQL Operations
Demonstrates creating tables and querying data using SQL.

```bash
node sql_check.js
```
