# ToonDB Rust SDK Examples

This directory contains examples for using the ToonDB Rust crate.

## Prerequisites

- [Rust](https://www.rust-lang.org/) installed (cargo).
- ToonDB crate: `toondb`

## Setup

Build the project:

```bash
cargo build
```

## Examples

### Basic Key-Value Operations
Demonstrates basic PUT and GET operations in Rust.

```bash
cargo run
```
